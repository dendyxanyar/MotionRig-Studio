# MotionRig Studio

MotionRig Studio is a lightweight interactive portfolio app built with React, Tailwind CSS, Framer Motion, and Lucide Icons.

It demonstrates a creative animation workflow for 2D character rigging, including body-part mapping, hidden overlap joints, prompt engineering, and visual rig preview.

## Features

- Interactive 2D rig preview
- Animated body-part layer map
- Prompt engineering toolkit
- Modern glassmorphism UI
- Fully responsive layout
- Lightweight and deploy-ready

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Use Case

Perfect for a creative developer portfolio, especially for animation, AI image workflow, Moho Pro rigging, and character production pipeline projects.
